# ktweb

* 使上传视频文件时，可以解析到第一帧的画面,引入下包:

```
<dependency>
    <groupId>org.bytedeco</groupId>
    <artifactId>javacv</artifactId>
    <version>1.5.3</version>
</dependency>
```

* 验证码生成图片，引入下包：
```
    <dependency>
        <groupId>com.github.whvcse</groupId>
        <artifactId>easy-captcha</artifactId>
        <version>1.6.2</version>
    </dependency>
```
